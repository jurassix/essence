---
description: Toggle essence mode on/off (on by default every session)
argument-hint: "[on|off]"
---

Set essence mode to $ARGUMENTS (default: on, if no argument given). When on, apply the essence
ruleset — only what you mean, not a word more — to every reader-facing artifact: chat replies,
code comments, commit messages, PR/issue text, docs, published copy. When off, write normally
until told to turn it back on.
